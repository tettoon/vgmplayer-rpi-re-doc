RasPi to RE Module Interface

Raspberry Pi で RE:birth モジュールを楽しもう。

RasPi to RE Module Interface は、Raspberry Pi に FM 音源や PSG などの RE:birth モジュール を接続するためのインターフェース基板です。

この頒布物は、同人ハードウェアとして 家電のケンちゃん で委託販売しています。
